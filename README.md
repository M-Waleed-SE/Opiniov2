# Opiniov2

A modern full-stack blogging platform built with the MERN stack.

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS, Redux Toolkit
- **Backend**: Node.js, Express, MongoDB, JWT
- **Deployment**: Railway (Backend), GitHub Pages (Frontend)

## Quick Start

### Backend
1. `cd BACKEND`
2. `npm install`
3. Set `MONGO_URI` and `CORS_ORIGIN` in `.env`
4. `npm run dev`

### Frontend
1. `cd CLIENT`
2. `npm install`
3. `npm run dev`

## Features
- Secure Auth (JWT)
- Article management (CRUD)
- Admin Dashboard
- Responsive Design & Dark Mode
- SEO Optimized Routing
